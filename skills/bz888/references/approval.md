# Approval

## Low-risk without extra approval

- Open a same-host public URL.
- Inspect a public book detail page.
- Inspect a public chapter page.
- Run a dry-run download manifest with `--json`.

## Requires approval

- Submit a search form.
- Execute a full-book download after dry-run review.
- Install or configure OCR dependencies.
- Change selectors or profile files based on new live evidence.

## Not allowed

- Solving or bypassing Cloudflare, CAPTCHA, login, permission, or risk-control surfaces.
- Submitting credentials, payment forms, verification forms, uploads, or unknown forms.
- Persisting raw cookies, auth headers, CSRF values, tokens, session ids, browser profiles, or equivalent sensitive material.
